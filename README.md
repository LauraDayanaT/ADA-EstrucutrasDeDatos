# Solución de Problemas de HackerRank (Punto 5)

Este repositorio contiene las soluciones en Python para los dos desafíos de programación competitiva propuestos en el Punto 5 del taller de Estructuras de Datos.

---

## Problemas Resueltos

### 1. CYCLE DETECTION
* **Enlace:** [HackerRank - Cycle Detection](https://www.hackerrank.com/challenges/detect-whether-a-linked-list-contains-a-cycle/problem)
* Descripción: Determina si una lista simplemente enlazada contiene un ciclo infinito en su estructura.
* Solución: Se implementó el Algoritmo de Floyd (Tortuga y Liebre) con dos punteros a diferentes velocidades, logrando una eficiencia de O(n) en tiempo y O(1) en espacio.

### 2. LARGEST RECTANGLE
* Enlace: [HackerRank - Largest Rectangle](https://www.hackerrank.com/challenges/largest-rectangle/problem)
* Descripción: Calcula el área del rectángulo más grande que se puede formar dentro de un histograma (serie de edificios alineados).
* Solución: Se utilizó una Pila Monótona Ascendente para procesar las alturas de forma lineal, optimizando el tiempo de ejecución a O(n).
